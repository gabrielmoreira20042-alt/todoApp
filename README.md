# To Do App

A modern and responsive To Do application built with Vanilla JavaScript.

## Features

- Add tasks with date and time
- Automatic sorting by deadline
- Overdue task highlighting
- LocalStorage persistence
- Smooth animations
- Responsive design

## Technologies Used

- HTML5
- CSS3 (Custom properties & responsive design)
- JavaScript (DOM manipulation & LocalStorage API)

## How to Run

1. Clone the repository
2. Open `index.html` in your browser

## Future Improvements

- Filter by "Today / Upcoming / Completed"
- Dark/Light mode toggle
- Drag & drop task reordering
- Notifications system
